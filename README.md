# AddisMenu 🍽️

A Django web application for discovering hotel menus in Addis Ababa, Ethiopia.

## Features

- 🏨 Hotel registration with admin approval workflow
- 📱 Menu management system for hotel owners
- 🔍 Search hotels by name or address
- 📍 Nearby hotel search using geolocation
- 🗺️ Google Maps integration for hotel locations
- 📸 Image upload for menu items
- 📱 Responsive Bootstrap 5 design

## Tech Stack

- **Backend:** Django 4.x
- **Database:** PostgreSQL
- **Frontend:** Bootstrap 5, HTML5, CSS3
- **Maps:** OpenStreetMap / Google Maps API
- **Authentication:** Django built-in auth
